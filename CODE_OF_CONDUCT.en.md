# Code of Conduct

> **한국어**: [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

This project adopts the **Contributor Covenant, version 2.1** as its Code of Conduct. By participating, you agree to uphold it.

The full, authoritative text is published by the Contributor Covenant project:

- English: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Other languages: <https://www.contributor-covenant.org/translations/>

A Korean rendition is also kept in this repository at [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).

## Scope

The Code of Conduct applies within all project spaces — the repository, issues, pull requests, and discussions — and when an individual is representing the project in public spaces.

## Reporting

To report unacceptable behavior, contact the maintainers at [opensource@sktelecom.com](mailto:opensource@sktelecom.com). All reports are reviewed and investigated promptly and fairly, and the reporter's privacy and security are respected.

Enforcement follows the Contributor Covenant 2.1 enforcement guidelines (Correction, Warning, Temporary Ban, Permanent Ban), as described in the full text linked above.
